# HopperMod — Forge

Changes hopper transfer speed and item amount per transfer via command.  
No OP level required — works directly in Singleplayer or on servers!

---

## Branches / Minecraft Versions

| Branch | Minecraft Version |
|--------|-----------------|
| `master` | 1.20.x |
| `1.21`  | 1.21.x |

Use the corresponding branch for the Minecraft version you want to build or play on. Some code differs between versions because of changes in Minecraft internals.

---

## Commands

| Command | Description |
|---|---|
| `/hopperspeed` | Shows the current setting |
| `/hopperspeed <1–6400>` | Ticks between transfers (Vanilla = 8, lower = faster) |
| `/hopperamount` | Shows the current setting |
| `/hopperamount <1–9999>` | Items per transfer (Vanilla = 1) |
